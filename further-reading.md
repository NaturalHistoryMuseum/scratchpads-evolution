## RDF Ontologies

- Scratchpads uses the SpeciesProfileModel ontology which has been deprecated by TDWG. An alternative is to use the GBIF [Taxon Descriptions](https://tools.gbif.org/dwca-validator/extension.do?id=gbif:Description) and [Description Type](https://rs.gbif.org/vocabulary/gbif/description_type.xml) extensions.

-   [SPM topic in GBIF forums](https://discourse.gbif.org/t/species-profile-model/115/7)


-   [GBIF DwC extension list](https://tools.gbif.org/dwca-validator/extensions.do)


- 	Shapes Constraint Language ([https://www.w3.org/TR/shacl/#introduction](https://www.w3.org/TR/shacl/#introduction)) as a way of describing what data can be stored in a given storage system.
